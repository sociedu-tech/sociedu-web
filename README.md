## 🚀 Chạy dự án (Run Locally)

**Prerequisites:** Node.js v18+

1. **Cài đặt thư viện:**
   `npm install`
2. **Cấu hình môi trường:**
   Sao chép `.env.example` thành `.env` và điền các thông tin cần thiết.
3. **Chạy dự án:**
   `npm run dev`
